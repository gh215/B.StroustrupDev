Глава 3. Objects, types and values!

cin - объект потока ввода в C++, используется для считывания данных с клавиатуры в программе.
безопасность типов - концепция, при которой компилятор не допускает некорректных операций с данными разных типов, предотвращая ошибки.
декремент -- - уменьшение значения переменной на 1 (например, x--).
значение - конкретное числовое или символьное содержимое переменной или константы.
имя - идентификатор, которым обозначена сущность (переменная, функция, класс и т.д.).
инициализация - присвоение начального значения переменной.
инкремент ++ - увеличение значения переменной на 1 (например, x++).
конкатенация - объединение двух строк в одну.
объект - экземпляр класса, структуры данных или другой сущности в программе.
объявление - определение типа и имени переменной, функции или другого объекта.
оператор - символ или комбинация символов, задающая операцию (например, +, -, =, ++).
операция - действие над одним или несколькими операндами с использованием операторов.
определение - задание реализации функции, класса или другого объекта.
переменная - именованная область памяти для хранения данных.
преобразование - изменение типа данных (например, из int в double).
присваивание = - операция, при которой переменной присваивается новое значение.
сужение - преобразование из одного типа данных в другой с возможной потерей информации.
тип - категория данных, определяющая диапазон значений и доступные операции (int, double, bool и др.).
