Глава 2. Hello, Word!

#include - директива препроцессора, позволяющая подключать заголовочные файлы (содержащие описания функций, классов и т.д.) к программе.
// - однострочный комментарий в C++. Все после // до конца строки игнорируется компилятором.
<< - оператор вывода в C++, используется с объектом cout для вывода данных в консоль.
cout - объект потока вывода в C++, используется для вывода данных в консоль.
IDE (Integrated Development Environment) - интегрированная среда разработки, предоставляющая редактор кода, компилятор, средства отладки и другие инструменты для разработки программ.
main() - обязательная функция, с которой начинается выполнение программы на C++.
библиотека - набор готовых функций, классов и других ресурсов, которые можно использовать в программе.
вывод - отображение данных программы, например, в консоли или окне.
выполняемый файл - файл, который содержит скомпилированный машинный код программы и может быть запущен.
заголовок (header file) - файл с расширением .h, содержащий описания функций, классов и других объектов.
инструкция - элементарное действие в программе, например, объявление переменной, вызов функции.
исходный код - текст программы, написанный программистом на языке программирования, понятном человеку.
комментарий - пояснения в исходном коде, игнорируемые компилятором. Комментарии нужны для документирования кода.
компилятор - программа, переводящая исходный код на машинный язык (объектный код).
объектный код - машинный код программы, полученный после компиляции исходного кода.
ошибка времени компиляции - ошибка в исходном коде, обнаруженная компилятором до запуска программы.
программа - набор инструкций, предназначенных для выполнения определенных задач на компьютере.
редактор связей (linker) - программа, объединяющая объектные файлы в единый выполняемый файл.
функция - именованный блок кода, выполняющий определенные действия. Может принимать данные на вход и возвращать результат.
