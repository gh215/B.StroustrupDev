Глава 4. Calculation

begin() - функция-член контейнеров в C++, возвращающая итератор на первый элемент.
else - ключевое слово для альтернативной ветви в условных инструкциях if-else.
lvalue (left value) - выражение, идентифицирующее разыменовываемый объект (переменная, элемент массива и т.д.).
push_back() - функция-член контейнеров в C++, добавляющая элемент в конец.
rvalue (right value) - временный неименованный объект, используемый как источник данных.
size() - функция-член контейнеров, возвращающая количество элементов.
sort() - функция для сортировки элементов контейнера по возрастанию.
выбор - операция, при которой выполняется одна из нескольких альтернатив на основе условия.
вывод - операция, при которой данные выводятся (отображаются) пользователю.
выражение - комбинация значений, переменных, операторов и функций, которая вычисляется.
вычисление - процесс определения значения выражения.
декремент -- - уменьшение значения переменной на 1.
инкремент ++ - увеличение значения переменной на 1.
инструкция - наименьшая исполняемая часть кода.
инструкция if - условная инструкция для выбора одной из двух ветвей в зависимости от условия.
определение - полная реализация функции, класса или другой сущности.
повторение - выполнение одних и тех же инструкций несколько раз.
разделяй и властвуй - подход для решения сложных задач путем разбиения их на более простые подзадачи.
условная инструкция - инструкция, выполнение которой зависит от определенного условия.
функция - именованная последовательность инструкций для решения определенной задачи.
функция-член - функция, определенная внутри класса.
цикл - инструкция для многократного выполнения блока кода.
цикл for - циклическая инструкция, повторяющаяся заданное количество раз.
vector - контейнер std::vector в С++, динамический массив.
абстракция - принцип сокрытия ненужных деталей и выделения главных концепций.
ввод - операция считывания данных, вводимых пользователем.
инструкция switch - инструкция для выбора одной из нескольких ветвей в зависимости от значения выражения.
итерация - одно выполнение цикла.
объявление - определение имени и типа объекта (переменной, функции и т.д.).
цикл for для диапазона - синтаксис for (: диапазон), итерирующий по диапазону.
цикл while - циклическая инструкция с предусловием.
